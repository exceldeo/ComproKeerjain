SI-Monitoring Sensor

Project ini merupakan sebuah Aplikasi Website (webapps) yang tersambung dengan Sensor Arduino. Tujuan dan fungsi utama Website ini yaitu untuk melakukan monitor terhadap status, nilai, dan data-data dari sensor yang ada sekaligus melakukan (on/off) terhadap sensor tersebut. Penjelasan Fitur-fitur tersebut dirincikan sbb:
  1. Halaman Login/Register
  2. Halaman Dashboard
    • Melihat data terakhir dari sensor
    • Melihat status sensor
    • Melakukan on/off terhadap sensor
  3. Halaman Grafik
    • User dapat melihat secara spesifik bagaimana perubahan data pada sensor yang ada yang disajikan dalam grafik
  4. Halaman Monitoring
    • User dapat melakukan
    • Export pdf : dimana data dari sensor dapat diunduh dalam format pdf
    • Import Excel : User juga dapat memasukkan data sensor melalui excel
  5. Halaman User
    • Menu tempat Admin dapat mengelola user yang ada
