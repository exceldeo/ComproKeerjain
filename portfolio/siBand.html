Management Checklist Terminal Bandara

Proyek ini berbasis Aplikasi Mobile yang memiliki fungsionalitas utama untuk menyimpan data pencatatan inspeksi kondisi fasilitas ruangan pada terminal-teminal yang ada pada sebuah bandara. Aplikasi ini terbagi menjadi dua role, yaitu pengguna dan admin. Berikut detail gambaran umum fitur per-role dan tangkapan layar aplikasinya:
  1. Halaman Login/Register (Pengguna/Admin)
  2. Halaman Role Admin
    • Pada sisi admin dapat melakukan penambahan data terminal, lantai pada terminal, dan ruangan dalam terminal. Selain itu, admin juga bisa mencetak rekapan laporan dari pengguna dalam bentuk data semacam excel dengan format csv.
  3. Halaman Daftar Laporan
  4. Halaman Detail Laporan
  5. Menu Cetak Excel dan Menu Utama Aplikasi Admin
  6. Halaman Daftar Lantai
  7. Halaman Daftar Ruangan
  8. Halaman Daftar Terminal
  9. Halaman Daftar Pengguna
  10. Halaman Role Pengguna
    • Secara umum, pada sisi pengguna memiliki fitur untuk melakukan pencatatan laporan kondisi dengan disertai foto dokumentasi, kemudian pengguna juga bisa melihat daftar laporannya.
  11. Halaman Daftar Lantai, Ruangan, dan Laporan
  12. Halaman Fasilitas Raungan
  13. Halaman Pelaporan
